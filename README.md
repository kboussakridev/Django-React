# 💰 Gestion des Dépenses - Fullstack Django + React

Bienvenue dans ce projet de **Gestion des Dépenses** ! C'est une application moderne et intuitive conçue pour t'aider à suivre tes finances au quotidien. On a allié la puissance de **Django** en backend et la réactivité de **React** en frontend pour une expérience fluide.

## 🌟 Pourquoi cette application ?
L'idée était de créer un outil simple où on ne perd pas de temps. Tu ajoutes tes revenus, tes dépenses, et l'application s'occupe de calculer ton solde et ton ratio de santé financière. Pas de chichis, juste l'essentiel avec un design qui claque.

## ✨ Fonctionnalités clés
- 🏎️ **Tableau de bord ultra-réactif** : Ton solde et tes statistiques se mettent à jour instantanément.
- 🛠️ **Gestion simplifiée** : Ajout et suppression de transactions en deux clics via une interface moderne.
- 🎨 **Design aux petits oignons** : Utilisation de **DaisyUI** pour un look épuré et **Tailwind CSS 4** pour la flexibilité.
- 📊 **Suivi des flux** : Visualisation claire de ce qui rentre et de ce qui sort.

## 🛠️ Coulisses Techniques
- **Backend** : Django & Django REST Framework (API robuste et sécurisée).
- **Frontend** : Next.js + React 19 + TypeScript (Performance et typage).
- **Style** : Tailwind CSS 4 & DaisyUI 5.
- **Icongraphie** : Lucide-react.

## 🚀 Prise en main (Installation Rapide)

C'est parti ! Pour lancer l'application sur ton poste local :

### 1. Préparer le Backend (Django)
Ouvre un terminal dédié au dossier `backend` :
```bash
cd backend
env/Scripts/activate
python manage.py runserver
```
> **Note** : Le serveur tournera par défaut sur [http://localhost:8000](http://localhost:8000).

### 2. Lancer le Frontend (Next.js)
Ouvre un second terminal pour le dossier `frontend` :
```bash
cd frontend
npm run dev
```
> **Note** : L'interface sera accessible sur [http://localhost:3000](http://localhost:3000).
